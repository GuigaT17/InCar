O InCAR é uma interface desenhada para ser usada no carro. É um painel que ajuda o motorista a fazer diversas tarefas enquanto dirige. Essa aplicacao foi desenvolvida para o projeto de IPM pelo grupo 040, constituido por Guilherme Teixeira 49021,

Permite acessar as configuracoes, bluetooth e wifi

Permite acessar o GPS
